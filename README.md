# 💰 Expense Tracker Full-Stack Application

A modern, full-stack web application to track daily expenses with a stylish user interface and smart budget management.

## ✨ Features
- **Shining Wavy UI**: Aesthetic and smooth glassmorphism design.
- **Budget Alert System**: Notifies you when expenses exceed your set budget.
- **Full CRUD Operations**: Add, View, and Manage your expenses easily.
- **Responsive Design**: Works perfectly on both desktop and mobile.

## 🛠️ Tech Stack
- **Backend**: Java, Spring Boot, MySQL (via XAMPP)
- **Frontend**: HTML5, CSS3 (Wavy UI), JavaScript
- **Tools**: VS Code, Git/GitHub

## 🚀 How to Run Locally
1. Clone this repository.
2. Open **XAMPP** and start Apache and MySQL.
3. Import the database (if applicable) or ensure your `application.properties` matches your local DB.
4. Run the Spring Boot application from VS Code.
5. Open `tracker.html` in your browser.
